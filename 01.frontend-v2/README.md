# 01.frontend-v2

이 디렉토리는 **개선형 채팅 UI**를 제공하는 프론트엔드 실습 코드로, API 경로 선택 등 데모 친화 기능을 포함합니다.

## 이 디렉토리 기준 구성

- `index.html`: 채팅 레이아웃과 API 경로 선택 UI
- `js/chat.js`: 메시지 전송/표시, 타이핑 인디케이터, 새 대화 초기화 로직
- `css/chat.css`: v2 채팅 화면 스타일
- `image/`: 사용자/어시스턴트 아바타 및 아이콘
- `nginx.conf`: Nginx 정적 서빙 설정
- `Dockerfile`: 정적 프론트 컨테이너 빌드
- `docker-build.sh`, `docker-push.sh`: 이미지 빌드/푸시 스크립트
- `k8s/`: 배포 매니페스트 템플릿 및 결과 파일

## 한 줄 요약

사용성 개선이 들어간 채팅 UI로 MCP 클라이언트 API 연동을 테스트하는 프론트 예제입니다.
